# Banking Loan Analysis ЁЯУК

## Overview

This project analyzes a **40,000-record banking loan dataset** using **Power BI** to evaluate loan performance, borrower behavior, repayment trends, and key financial metrics. The project demonstrates the complete Power BI workflow, including data transformation with Power Query, DAX calculations, interactive dashboard development, business reporting, and presentation creation.

---

## Dataset

* **Dataset Size:** 40,000 loan records
* **Domain:** Banking & Financial Services
* **Key Features:**

  * Loan Status
  * Loan Amount
  * Funded Amount
  * Total Payment Received
  * Interest Rate
  * DTI Ratio
  * Employment Length
  * Employment Title
  * Loan Purpose
  * Home Ownership
  * Annual Income
  * Installment Amount
  * Credit Grade
  * Issue Date
  * Last Payment Date
  * Member ID

## Project Workflow

### 1. Data Loading

* Imported the 40K loan dataset into Power BI.
* Verified data types and column formats.

### 2. Power Query (Data Transformation)

* Cleaned and transformed data.
* Removed duplicate records.
* Handled missing values.
* Renamed columns.
* Corrected data types.
* Created necessary derived fields.

### 3. Data Modeling

* Built relationships between tables (if applicable).
* Optimized the data model for reporting.

### 4. DAX Measures

Created measures for:

* Total Loan Applications
* Total Funded Amount
* Total Amount Received
* Average Interest Rate
* Average DTI Ratio
* Good Loan Percentage
* Bad Loan Percentage
* Loan Status KPIs

### 5. Power BI Dashboard

Designed an interactive dashboard including:

* KPI Cards
* Loan Application Overview
* Good vs Bad Loan Analysis
* Funded Amount vs Amount Received
* Loan Status Distribution
* Average Interest Rate
* Average DTI Ratio
* Loan Purpose Filters

### 6. Reporting

Prepared a business report highlighting:

* Loan portfolio performance
* Repayment trends
* Risk analysis
* Business insights
* Recommendations

### 7. Presentation

Created a professional presentation using **Gamma** covering:

* Project overview
